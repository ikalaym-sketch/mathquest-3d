# MathQuest 3D — 架構記錄 v0.8.0（程序化音樂與音效）

## 方法說明：為何用「程序化合成」而非音檔
- 無法呼叫文字轉音樂模型產 mp3；但那對網頁遊戲也非最佳解。
- 採 **Web Audio API 即時合成**：以程式定義音階/和弦進行/配器/音色，瀏覽器執行期產生真實聲音。
- 優點：零音檔（GitHub Pages 友善）、離線可用、體積僅 +~12KB、每主題皆為實際作曲。

## 音樂設計（每個主題的作曲方向）
- **村莊 village**：溫暖田園大調(I–vi–IV–V, maj7)，中速 96，三角波柔和主奏 → 安定居家感
- **牧場 farm**：明亮五聲大調，輕快 112，撥弦方波 + 輕鼓 → 農忙愉悅
- **平原/浮空島 heroic**：英雄大調 122 + 鼓 → 冒險感
- **荒漠/遺跡 exotic**：Phrygian Dominant 異國音階 100 + pad → 神秘
- **極地 cold**：小調 82 慢速 + pad → 寒冷孤寂
- **沼澤/幽暗 eerie**：小調含減和弦 76 稀疏 pad → 陰森
- **火山/混沌 intense**：小調 142 驅動貝斯 + 鼓 → 危險急迫
- **蘑菇林 whimsical**：Lydian 118 → 奇幻俏皮
- **水晶洞 ethereal**：大調加九 88 + shimmer pad → 空靈
- **Boss/試煉 boss**：和聲小調 150 全配器 + 緊張 → 決戰
- **數學挑戰 math**：Dorian 104 極簡（滴答感）→ 專注
- **氣球 balloon**：五聲大調 126 → 輕鬆小遊戲

## 系統架構
- `src/audio/AudioEngine.js`：AudioContext + master/music/sfx 匯流排 + 回授延遲殘響 + `playNote`(ADSR 包絡防爆音)/`playNoise`/`mtof`
- `src/audio/themes.js`：12 主題資料（root/scale/progression/voices/timbre/tempo）+ 生態系→主題對應
- `src/audio/music.js`：前瞻式排程器（lookahead 0.1s/25ms tick），依配器演奏 bass/arp/lead/pad/drums(kick/snare/hat)，主題可切換
- `src/audio/sfx.js`：13 種合成音效
- `AudioController`：首互動解鎖音訊（瀏覽器政策），依 store 狀態選主題（覆蓋序：氣球>Boss>試煉>數學>場景/生態系）
- `AudioToggle`：左下音樂/音效靜音鈕

## 音效串接（13 種）
attack(揮擊)、hit(命中/爆擊)、enemyDeath、playerHurt、pickup、dodge(翻滾)、click、correct(答對琶音)、wrong(答錯減音)、levelUp(天賦解鎖號角)、seal(Boss封印華麗和弦)、harvest(收成)、chest(開寶箱)
- 已接入：Player(攻擊/翻滾)、Monster(命中/死亡/使玩家受傷)、MathModal & Balloon(對/錯)、FarmScene(收成)、Chest(開箱)、Boss(封印成功)、Journal(天賦解鎖)

## 驗證
- `npm run build` 無警告；未使用 import 掃描 0 筆
- 模擬 AudioContext 測試：13 SFX 全通過、12 主題設定+排程無錯、10 生態系對應、和弦型別全有效
- 推進時間實跑排程器（boss/farm/ethereal/village）4 秒/多次：全配器(鼓/pad/琶音/主旋律) 0 錯誤
- 瀏覽器政策：AudioContext 於首次 pointer/key/touch 才啟動（符合自動播放限制）

## 使用說明
- 進遊戲點一下畫面即解鎖音訊；音樂隨場景/生態系自動切換
- 左下 🎵/🔊 可分別靜音音樂與音效
