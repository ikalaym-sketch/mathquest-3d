## 通行系統限制

- 敵人目前使用共享幾何投影與滑動回退，不是正式 NavMesh／尋路網格。
- 高地由矩形平台及斜坡道路組成，尚未替換為最終地形 GLB／Heightfield。
- 本版範圍為八個正式區域；村莊、農場與森林遺跡仍保留各自既有地形系統。
- 游泳尚未完成，因此玩家進入一般水域會安全退回最近可行走位置。
