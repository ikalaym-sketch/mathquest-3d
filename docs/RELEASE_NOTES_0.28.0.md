# MathQuest 3D v0.28.0｜Village Bonds & Guardian Companions

## 基準

- 唯一開發基準：`v0.27.0 Life Simulation Core Truth`。
- 本版合併原規劃階段 C／D，不回退、不建立平行 Runtime。

## 階段 C｜村莊關係與生活循環

- 建立 10 名具名居民 Canonical Profile：姓名、職業、個性、住家、工作地、休閒地、雨天位置、休息日、禮物偏好與關係事件。
- 每位居民至少 30 條條件式對話來源，依時間、天氣、星期、季節與關係階段組合。
- 每位居民具有 3 段多步驟關係事件，共 30 段；事件完成具防重複與永久記憶旗標。
- 好感改為 0～100 百分制，支援每日首次交談、每日送禮限制、喜愛／喜歡／一般／討厭四級反應。
- 角色日程依星期、時間及天氣切換工作、休閒、返家與避雨；功能 NPC 休息日仍可交談，但服務不開放。
- 農場出貨會累積市場等級並改變村莊商店貨架；商品依市場等級、星期及出貨紀錄輪替。
- 每遊戲週產生 4 項村莊委託，交付後連動金幣、居民好感及共同記憶。
- 四季各建立一場村莊活動：春芽農產品市集、夏日夥伴野餐會、秋收數學祭、冬夜水晶燈節。

## 階段 D｜動物守護夥伴

- 舊 Pet 裝備 Runtime 停止；新存檔由狸貓、兔子、狐狸三選一，舊存檔依原 Pet 安全映射。
- 建立 8 隻正式夥伴 Profile：狸貓、兔子、狐狸、小貓、水獺、松鼠、小雞、小熊貓。
- 每隻夥伴具有生活、探索、學習、戰鬥四類能力，以及唯一取得條件、喜好、個性、等級、經驗、好感及每日互動。
- 5 隻非初始夥伴分別由居民關係、區域 Boss、農場動物產出及森林事件取得，不使用抽卡。
- 8 個低多邊形工程 GLB 均具 Canonical 節點、Collider、4 個 Socket 與 8 組有效動畫 Clip。
- 夥伴會在地面跟隨、游泳、問候、開心、休息；非同行夥伴會返回農舍旁休息區。
- 八隻夥伴均至少有一項能力接入實際 Runtime：材料掉落、學習提示／回血、移速、首次交談、水中氧氣／泳速、種子保留、動物品質、加工縮時。
- 新增初始三選一、夥伴名冊、同行切換、每日撫摸／玩耍／休息與取得提示。

## 工程原則

- 村莊、關係、經濟、活動、夥伴、取得與 3D Runtime 分檔管理。
- 主 Store 透過 `villageSlice`／`companionSlice` 接入，避免繼續膨脹單一功能區塊。
- 存檔 schema 升級；舊 Pet 映射後清除裝備欄 Pet，避免新舊雙 Runtime。
- 測試指令純讀取；證據只由明確 evidence export 指令寫入。

## 尚未冒充完成

- 8 個夥伴 GLB 為可替換工程低模，不是最終委製角色美術。
- 村民仍使用共用角色 Rig 與職業配色，尚未完成每人專屬臉型、服裝與語音。
- 季節活動目前完成資料、交付、好感與報酬閉環，尚未製作大型節慶專屬場景演出。
- 容器 Chromium 受 `URLBlocklist=["*"]` 管理政策限制，實際多裝置 WebGL 畫面仍為 BLOCKED，不標示 PASS。
