# v0.23.0 Destructive QA

## 已執行破壞性檢查

1. 全圖單一 Collider 不得重新掛載。
2. 水面不得建立可站立的隱藏 Floor Collider。
3. 峽谷裂縫必須使用獨立非水面 Runtime。
4. 道路任一採樣點若落入水域，必須同時位於有效橋面。
5. 每個地表矩形中心與四角不得覆蓋水域。
6. 32 棟結構以 footprint 網格採樣，不得侵入水域。
7. 80 個一般／精英／Boss 出生點必須位於可行走表面。
8. 玩家 grounded 必須使用目前地形採樣高度。
9. 玩家必須保存最後安全位置供水域／裂縫復原。
10. 一般怪物與 Boss 必須使用共享通行投影。
11. 道路坡度不得超過 18°。
12. 路徑無解時不得直接恢復穿水直線。

## 最危險的三個後續破綻

- 加入游泳後若未建立上岸檢查，玩家可能永久困在深水。
- 將程序地形替換為 GLB 時若 Collider 與視覺網格分離，可能再次出現懸空或穿模。
- 敵人數量提高後，現有幾何投影不是完整 NavMesh，複雜窄路可能產生群聚堵塞。
