# v0.22.0 強制紅隊測試

## 已驗證的破壞性情境

1. **舊存檔升版**：已探索子區、已完成機關、已使用 Socket、已擊敗 Boss 會回填主線；精英沒有唯一舊證據時不會誤判。
2. **重複 Signal**：已完成任務再次收到相同擊殺／互動 Signal，不會重複發放 Gold。
3. **場景切換中關閉題目**：數學與 Boss 封印 Callback 含 `sceneEpoch`，舊場景回呼無法寫入新場景。
4. **Boss 線型攻擊轉向**：測試正交方向、命中方向、外部方向及圓形邊界，預警幾何與命中判定一致。
5. **髒 UI 設定**：不合法文字大小、色覺模式會回復 Canonical 預設，合法 HUD 選項保留。

## 仍可能造成交付失敗的 3 個破綻

1. **實際裝置 HUD 重疊**：靜態 CSS 與 Build 可通過，但因環境封鎖，390×844、1024×768、1440×900 的真實 WebGL 疊層尚未人工驗收。
2. **水域與橋梁碰撞語意**：現有 BaseTerrain 仍可讓角色穿越水域／裂隙，會破壞部分場景敘事與路線設計；此項依指示延後封版修正。
3. **語音引擎差異**：部分 Android WebView 或桌面瀏覽器可能沒有繁體中文 Voice，屆時只會保留字幕，不會有語音；不會崩潰，但功能體驗會降級。
