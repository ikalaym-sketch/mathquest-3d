# MathQuest 3D v0.22.0｜戰鬥、兒童 UI、故事學習整合版

本版本以 v0.19.0 為唯一基準，合併完成原規劃的 v0.20、v0.21、v0.22。沒有回退舊工程，也沒有以資料宣告冒充 Runtime 接線。

## v0.20｜Boss、精英與試煉塔重建

- 8 名區域 Boss 與 10 名試煉塔里程碑 Boss 全部使用三階段戰鬥 Profile。
- 每個階段包含移動模式、技能循環、700ms 以上可視預警、傷害範圍與反擊窗口。
- 線型預警的畫面旋轉與命中判定共用 `CombatGeometryService`，避免視覺與傷害範圍漂移。
- Boss 封印由「全對才成功」改為 Canonical 寬容門檻；高樓層才逐步提高要求。
- 8 種區域精英加入護盾層、週期脈衝與低血量強化，不再只是高血量一般怪。
- 試煉塔建立 10 套正式空間配置、至少 7 種不同輪廓及 10 種房型。
- 100 層使用戰鬥、耐力、群落、精英、陷阱、護送、場景序列、時序機關、寶物、休息及 Boss 流程。
- 場景式塔樓機關直接在 3D 世界操作，不再自動開啟共用 Math Modal。
- 新增 `sceneEpoch` Callback Cancellation；切換場景會清除舊數學題、Boss 封印、NPC 對話與函式型暫態回呼。

## v0.21｜兒童 UI Design System

- 建立統一淺色兒童遊戲 Design Tokens、卡片、彈窗、邊框、陰影與字級基準。
- 建立 `HUDLayoutManager`，依桌機、平板、手機、直向、橫向切換版位。
- 支援瀏海及系統導覽列安全區 `safe-area-inset-*`。
- 支援左右手模式、HUD 大小、文字大小、精簡 HUD、小地圖開關。
- 支援減少動態、高對比及三種色覺輔助模式。
- 主要操作入口改用共用 SVG `GameIcon`，降低 Emoji 作為功能圖示的依賴。
- 主要玩家操作、世界地圖、商店、背包、鐵匠、返回入口與載入錯誤已統一繁體中文。
- 語音引導已接入裝置內建 `SpeechSynthesis`，以 `zh-TW` 朗讀世界提示及 NPC 對話；不支援時安全略過。
- 字幕偏好會控制世界提示文字，語音與字幕可分開使用。

## v0.22｜故事、NPC 與學習融合

- 建立 8 章正式區域主線與 40 條任務，每章固定包含探索、場景機關、結構 Socket、精英及 Boss。
- 正式順序：星光村外圍 → 農業平原 → 風之高地 → 水晶湖 → 蘑菇濕地 → 日光峽谷 → 雪光谷 → 機械遺跡。
- 新存檔只開放森林遺跡與星光村外圍；完成章節才解鎖下一區並顯示具體故事原因。
- 舊版已開放的區域不會被反向鎖定。
- 舊存檔會依探索、機關、Socket 與 Boss 證據回填任務，避免要求重做；精英因舊版無唯一擊殺證據，不會誤判完成。
- NPC 對話會記錄談話次數、好感、最近話題、相遇及談論章節記憶。
- 村長、商人、鐵匠、木匠具有正式 Profile；八區引導員使用統一的區域 Profile 契約。
- 學習報告將傳統題目與場景式操作分開統計，提供家長可理解的正確率與加強建議。
- 場景式學習目前涵蓋風向角度、溫度、陣列乘法、序列、對稱、長度、倍數節奏及齒輪時序。

## 正式統計

- 區域 Boss Profile：8。
- 試煉塔 Boss Profile：10（由 Canonical Boss Profile 衍生）。
- Boss 階段：54。
- 精英 Profile：8。
- 試煉塔場景主題：10。
- 試煉塔樓層：100。
- 塔樓房型：10 個 Canonical Variant，Runtime 包含 11 個顯示 Archetype（含 Boss）。
- 主線章節：8。
- 正式任務：40。
- UI／可及性設定：10。

## 驗證界線

本版完成 v0.20-v0.22 的工程功能，但不宣稱完成最終委製美術或多裝置人工畫面驗收。Chromium 仍受到執行環境 `URLBlocklist=["*"]` 阻擋，因此 24 個桌機／平板／手機 WebGL 案例標示 BLOCKED，不標示 PASS。
