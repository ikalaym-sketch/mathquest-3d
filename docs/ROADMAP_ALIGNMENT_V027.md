# v0.27 藍圖對齊檢查

## 原藍圖

`v0.27` 原規劃為 Performance、Physical Device QA 與 Final Sealing。

## 本次實際決策

使用者在 v0.26 後明確要求先完成改善藍圖的階段 A／B，因此版本用途調整為：

- 階段 A：裝備／道具 Runtime 語意真實化。
- 階段 B：農務操作、品質商品、動物照顧、四季與經濟。

此調整不是漏做或宣稱原封版已完成；效能、實機及歷史封版工作順延至後續版本。

## 遺漏檢查

- 裝備／道具能力覆蓋：已納入。
- 品質商品與舊存檔遷移：已納入。
- 農具／體力／整地：已納入。
- 四季、動物照顧、加工時間、農莊施工：已納入。
- 正式村民關係與守護夥伴：不屬 A／B，保留於 C／D。
- 實機 WebGL、最終美術與歷史封版：明確延後，未隱藏。
