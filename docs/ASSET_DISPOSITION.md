## 規則

- Runtime 僅能以 `assetId` 查詢，不得直接保存或傳遞 `/models/...` URL。
- `dist/` 是建置輸出，不計入 Canonical 資產數。
- 狀態只能由 `src/data/assetDispositionContract.js` 決定。
- 新增資產必須先加入正式 Recipe、Contract Validator 與 Canonical Registry。
